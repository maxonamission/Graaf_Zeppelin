# S06-01: Voorbeeldvragen aanbieden

**Epic:** EPIC-06 AI-assistent bruikbaar maken
**Status:** 🔲 Backlog
**Prioriteit:** Hoog

## User Story

**Als** beleidsmedewerker
**Wil ik** klikbare voorbeeldvragen zien in de chat-interface
**Zodat** ik weet wat voor vragen ik kan stellen en snel op weg ben

## Acceptatiecriteria

- [ ] 5-8 voorbeeldvragen zichtbaar bij lege chat
- [ ] Vragen zijn domeinrelevant (sportdeelname, beleid, interventies)
- [ ] Klikken op een vraag stuurt deze als chat-bericht
- [ ] Voorbeelden verdwijnen na eerste interactie
