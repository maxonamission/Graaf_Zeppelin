# S06-02: Automatische factor-selectie

**Epic:** EPIC-06 AI-assistent bruikbaar maken
**Status:** 🔲 Backlog
**Prioriteit:** Hoog

## User Story

**Als** beleidsmedewerker
**Wil ik** dat de app zelf de relevante factoren selecteert op basis van mijn vraag
**Zodat** ik niet handmatig factoren hoef aan te vinken

## Beschrijving

Huidige situatie: gebruiker moet handmatig factoren aanvinken voor context.
Gewenst: de app analyseert de vraag en selecteert automatisch relevante factoren.

## Acceptatiecriteria

- [ ] Vraag wordt geanalyseerd op trefwoorden / domeinen
- [ ] Relevante factoren worden automatisch geselecteerd
- [ ] Gebruiker kan selectie handmatig aanpassen als dat nodig is
- [ ] Bij onduidelijke vragen: suggestie tonen in plaats van raden
