# S03-04: Chat interface — frontend met factorselectie

**Epic:** EPIC-03 AI-assistent (LLM-integratie)
**Status:** ✅ Done
**Module:** `app/templates/reasoning.html`

## Beschrijving

Chat-interface waarmee gebruikers in natuurlijke taal vragen stellen
over het causale model, met factorselectie en LLM-configuratie.

## Wat is gebouwd

- Multi-turn conversatie-interface
- LLM-provider selectie (OpenAI/Anthropic)
- API-key invoer (session-only)
- Factor-selectie via checkboxes
- Markdown-rendering van antwoorden
